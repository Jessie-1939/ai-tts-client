# AI-TTS Client 接入说明

本文档把当前项目的接入现状说清楚：**AI-TTS Client 目前只有一个供应商接入，就是火山引擎（豆包）的语音合成 API**。

## 核心摘要

- **供应商与体验**：当前仅接入 **火山引擎（豆包）语音合成大模型 V3** 的能力，主要使用其 **2.0 版本模型** 作为音源示例。
- **流式能力**：火山引擎的 V3 系列支持流式返回音频，因此插件可以在合成过程中尽快开始播放，减少等待感。
- **费用说明**：调用语音合成 API 会产生费用，通常按字符计费，费用由用户自行承担。
- **当前局限**：目前**不支持其他平台**。后续会根据需求考虑扩展到 **Azure、阿里云、腾讯云** 等平台。

## 用户需要做什么

使用此插件前，用户需要先在云服务控制台完成注册、开通服务并获取密钥，然后在插件内配置。

1. **注册与认证**：访问 [火山引擎官网](https://console.volcengine.com) 注册账号并完成实名认证。
2. **开通服务并获取密钥**：在 **API Key 管理** 页面创建并复制密钥。
3. **配置插件**：将 API Key 和所选音色填入插件配置页面即可使用。

> ！！！API Key 存储在用户本地浏览器中，不会上传到项目服务器。

## 相关文档与项目状态

- **开发文档**：更多技术细节可查阅官方的 **语音合成大模型 V3 文档**。当前插件使用的模型资源 ID 为 `seed-tts-2.0`。
- **开源项目**：项目全部代码已在 GitHub 开源，任何人都可以自行查阅、修改和分发。

## 接口与实现说明

当前后台实现使用的是 V3 的 HTTP 流式请求方式，核心请求特征如下：

- 接口地址：`https://openspeech.bytedance.com/api/v3/tts/unidirectional`
- 请求方式：HTTP `POST`
- 认证头：`X-Api-Key`
- 资源头：`X-Api-Resource-Id`
- 请求 ID：`X-Api-Request-Id`

### 请求体摘要

```json
{
  "user": { "uid": "ai-tts-extension-xxx" },
  "req_params": {
    "text": "待合成文本",
    "speaker": "音色 ID",
    "audio_params": {
      "format": "mp3",
      "sample_rate": 24000,
      "speech_rate": 0,
      "loudness_rate": 0
    }
  }
}
```

### 说明

- 当前项目的音色选择来自火山引擎 2.0 音色列表
- 当前实现已经完成本地配置与流式播放链路
- 如果后续接入 Azure / 阿里云 / 腾讯云，需要同步更新请求头、鉴权方式和音色映射
