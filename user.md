# AI-TTS Client 用户指南

> 这是给新用户看的快速上手说明。先看这份，再看 `README.md`、`PRIVACY.md` 和 `API接入.md`。

---

## 设计现状

- 本插件是一个**纯浏览器端工具**，没有后端服务器。
- 支持**选中网页文本 → 右键菜单 → 语音合成**。
- 内置流式播放器，支持播放 / 暂停、进度拖拽、速度调节。
- 配置界面可填写 API Key、选择音色，所有配置都保存在本地浏览器存储中。
- **当前仅支持火山引擎（豆包）语音合成 API V3**，使用 `seed-tts-2.0` 资源 ID 和 2.0 音色列表。
- 未来可能扩展其他供应商，但当前版本只有这一家。

---

## 用户需要做什么

1. **注册火山引擎账号**并完成实名认证。  
   → [火山引擎官网](https://console.volcengine.com/)
2. **开通语音合成服务**，获取 API Key。  
   → 控制台 → 语音技术 → API Key 管理 → 创建 API Key
3. **安装本插件**（从 Edge 商店或手动加载）。
4. 点击插件图标，填入 **API Key** 并选择音色。
5. 在任意网页选中文字，右键 → “朗读选中文本”。

### 你会在插件里看到什么

- 统计卡片：今日合成 / 历史总计
- API Key 区域：输入、显示 / 隐藏、修改
- 音色选择：2.0 音色下拉列表
- 操作按钮：测试连接、保存设置
- 底部链接：获取 API Key、使用文档、GitHub

## 数据与隐私（重要）

### 本插件不收集、不存储、不上传任何用户数据

1. **所有语音合成请求均直接从你的浏览器发送至火山引擎 API**  
   插件本身没有任何“云端中转服务”或“代理服务器”。你选中的文本、你配置的 API Key，全部由你的浏览器直接打包发给 `openspeech.bytedance.com`。  
   ✅ 作者看不到你的文本内容  
   ✅ 作者看不到你的 API Key  
   ✅ 作者没有服务器，也不记录任何日志

2. **本地存储仅用于保存你的配置**  
   插件使用 Chrome 扩展的 `storage.local` 机制保存以下信息：  
   - 你填入的 API Key（仅保存在你的电脑上，不上云）
   - 你选择的音色 ID
   - 本地统计的字符用量（今日 / 历史，仅用于你自己查看）  
   这些数据不会被同步到任何云端，也不会通过任何网络请求发送给作者。

3. **你与火山引擎之间的数据交互由你自行负责**  
   当你使用插件调用语音合成时，你实际上是在直接调用火山引擎的公开 API。你的文本内容会被发送到火山引擎服务器进行合成。火山引擎会按字符计费，费用从你的账户扣除。  
   你应当阅读并遵守 [火山引擎语音合成服务协议](https://www.volcengine.com/docs/6561/1257584) 及[隐私政策](https://www.volcengine.com/legal/privacy)。插件作者不参与、不干预、不记录你与火山引擎之间的任何通信。

4. **没有“后门”，没有“统计数据回传”**  
   本插件不会向任何第三方（包括作者）发送任何使用统计、错误报告、设备信息或用户行为数据。

5. **你对自己的 API Key 和费用负全责**  
   请妥善保管你的 API Key，不要分享给任何人。插件不会限制你每日合成字数，也不会帮你预估费用——因为你用的是自己的账户、自己的钱。  
   建议你在火山引擎控制台设置“费用预警”和“每日额度上限”。

> 如果你不相信，请自行阅读[开源代码](https://github.com/%E4%BD%A0%E7%9A%84%E4%BB%93%E5%BA%93%E5%9C%B0%E5%9D%80)。代码里没有任何网络请求指向非火山引擎的域名。

## 使用前你要知道的几件事

- 这不是“点一下就自动什么都给你配好”的工具
- 你需要自己有火山引擎账号、API Key 和可用额度
- 音色权限、额度、计费、服务状态都由火山引擎控制
- 如果 API 调用失败，先看官方文档，再看控制台，再看插件设置
- 这份工具是开源的，适合能自己动手的人
