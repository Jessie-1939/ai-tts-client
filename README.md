## 📦 安装教程

我们提供了两种安装方式，你可以根据需要选择。

### 方法一：通过 Microsoft Edge 插件商店安装（推荐）

1.  打开 Microsoft Edge 浏览器。
2.  点击右上角的“...”(更多) 菜单，选择“扩展”，或在地址栏直接访问 `edge://extensions/`，然后点击“**获取 Microsoft Edge 扩展**”[reference:0]。
3.  在 Edge 插件商店搜索框中，输入 **AI TTS Client** 进行搜索。
4.  在搜索结果中找到本插件，点击“**获取**”按钮。
5.  在弹出的权限确认窗口中，点击“**添加扩展**”即可完成安装[reference:1]。

> 💡 **直达链接**：你也可以直接点击 [**这里**](你的Edge商店链接) 访问插件在 Edge 商店的官方页面，一键安装。

### 方法二：开发者模式安装（用于开发或调试）

> **注意**：此方法主要适用于开发者对插件进行本地调试和开发，不建议普通用户使用。
...

1. **下载插件包**
   - 解压本压缩包到本地文件夹（例如 `D:\ai-tts-client`）

2. **打开Edge扩展管理页面**
   - 在地址栏输入：`edge://extensions/`
   - 或者：设置 → 扩展 → 管理扩展

3. **开启开发者模式**
   - 打开页面左下角的 **「开发人员模式」** 开关

4. **加载插件**
   - 点击 **「加载解压缩的扩展」**
   - 选择刚才解压的插件文件夹
   - ✅ 安装完成！插件图标会出现在工具栏

## 🔑 API 密钥获取

插件当前**只接入火山引擎的语音合成 API**，并以其 **语音合成大模型 V3 / 2.0 系列** 作为唯一音源示例。火山引擎的 V3 能力支持流式返回音频，因此可以在生成过程中边返回边播放，等待感更低。

**注意：API Key 只保存在用户本地浏览器中，不会上传到我们的服务器。**

使用前需准备 API 密钥：

### 步骤1：注册火山引擎账号
1. 访问 [火山引擎官网](https://www.volcengine.com/)
2. 注册并完成实名认证

### 步骤2：开通语音合成服务
1. 进入 [语音合成控制台](https://console.volcengine.com/speech/tts)
2. 点击 **「创建应用」**
3. 选择 **「大模型语音合成」**
4. 填写应用名称并提交

### 步骤3：获取密钥信息
1. 在应用详情页获取：
   - **API Key**：新版控制台使用的访问密钥
2. 复制 API Key 并填入插件配置界面

> 💡 **提示**：新用户有免费额度，具体以火山引擎官方定价为准

### 当前局限

- 目前**不支持其他平台**，包括但不限于 Azure、阿里云、腾讯云等
- 后续会根据用户反馈考虑逐步增加多供应商接入
- 当前默认音源仍以火山引擎为准，其他平台支持不在本阶段范围内

## 🔒 数据与隐私说明

### 本插件不收集、不存储、不上传任何用户数据

1. **所有语音合成请求均直接从你的浏览器发送至火山引擎 API**  
   插件本身没有任何云端中转服务或代理服务器。你选中的文本、你配置的 API Key，都会由浏览器直接发送到 `openspeech.bytedance.com`。  
   ✅ 作者看不到你的文本内容  
   ✅ 作者看不到你的 API Key  
   ✅ 作者没有服务器，也不记录任何日志

2. **本地存储仅用于保存你的配置**  
   插件使用 Chrome 扩展的 `storage.local` 机制保存以下信息：  
   - 你填入的 API Key（仅保存在你的电脑上，不上云）  
   - 你选择的音色 ID  
   - 本地统计的字符用量（今日 / 历史，仅用于你自己查看）  
   这些数据不会同步到任何云端，也不会通过任何网络请求发送给作者。

3. **你与火山引擎之间的数据交互由你自行负责**  
   当你使用插件调用语音合成时，你实际上是在直接调用火山引擎的公开 API。你的文本内容会被发送到火山引擎服务器进行合成。火山引擎会按字符计费，费用从你的账户扣除。  
   你应当阅读并遵守 [火山引擎语音合成服务协议](https://www.volcengine.com/docs/6561/1257584) 及隐私政策。插件作者不参与、不干预、不记录你与火山引擎之间的任何通信。

4. **没有“后门”，没有“统计数据回传”**  
   本插件不会向任何第三方（包括作者）发送任何使用统计、错误报告、设备信息或用户行为数据。

5. **你对自己的 API Key 和费用负全责**  
   请妥善保管你的 API Key，不要分享给任何人。插件不会限制你每日合成字数，也不会帮你预估费用——因为你用的是自己的账户、自己的钱。建议你在火山引擎控制台设置“费用预警”和“每日额度上限”。

> 如果你不放心，请直接阅读本项目的开源代码。代码中没有任何指向非火山引擎域名的网络请求。

## 🚀 使用说明

### 基础使用
1. **选中文本**：在任意网页选中要朗读的文字（最长5000字）
2. **右键菜单**：点击「AI-TTS → 朗读选中文本」
3. **等待生成**：插件显示优雅的加载动画
4. **开始播放**：底部出现极简悬浮条，自动开始播放

### 播放器操作
- **播放/暂停**：点击中央大按钮
- **进度跳转**：直接点击进度条或拖拽
- **快进/后退**：点击 ⏪ ⏩ 按钮（±5秒）

### 配置界面
点击浏览器工具栏的插件图标，打开配置面板：
- **API配置**：设置 API Key
- **音色选择**：15+ 种高品质音色任选
- **测试连接**：一键验证 API 是否配置正确
- **主题切换**：🌙 深色 / ☀️ 浅色主题

## ❓ 常见问题

### Q: 提示"请先配置API密钥"
A: 点击插件图标，在配置界面填入已启用的 TTS API Key

### Q: 语音生成失败怎么办？
A: 1. 检查网络连接 2. 确认 API Key 正确 3. 检查所选提供商的额度与权限

### Q: 支持哪些浏览器？
A: 完美支持 Edge、Chrome、Brave 等 Chromium 内核浏览器

### Q: 文本长度有限制吗？
A: 单条最长 5000 字符，超过会自动截断

### Q: 生成的音频保存在哪里？
A: 点击下载按钮会保存到浏览器默认下载目录
